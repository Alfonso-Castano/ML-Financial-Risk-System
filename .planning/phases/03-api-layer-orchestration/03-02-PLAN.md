**backend/api/routes.py** — Thin route handlers:

Create an `APIRouter` with two endpoints. Routes do ONE thing: receive request, call predictor, return response. No business logic, no torch imports, no feature engineering.

```python
from fastapi import APIRouter, Request
from backend.api.schemas import PredictRequest, PredictResponse, HealthResponse

router = APIRouter()

@router.post("/predict", response_model=PredictResponse)
async def predict(body: PredictRequest, request: Request) -> PredictResponse:
    result = request.app.state.predictor.predict(body)
    return result

@router.get("/health", response_model=HealthResponse)
async def health(request: Request) -> HealthResponse:
    result = request.app.state.predictor.health()
    return result
```

That's it. No additional logic. Routes are thin.

**backend/main.py** — FastAPI application:

1. **Lifespan context manager** using `contextlib.asynccontextmanager`:
   - On startup: instantiate `Predictor()` and store in `app.state.predictor`
   - Predictor constructor loads model, scaler, metrics (fail-fast if files missing)
   - On shutdown: nothing to clean up (yield with no teardown)

2. **Custom 422 error handler** — override `RequestValidationError` to return `{"error": "message"}` format instead of Pydantic's verbose structured errors:
   - Import `RequestValidationError` from `fastapi.exceptions`
   - Flatten error list: for each error, join the `loc` tuple with " -> " and append the `msg`
   - Join all messages with "; " and return as `{"error": joined_string}`
   - Return `JSONResponse(status_code=422, content={"error": ...})`

3. **Include router**: `app.include_router(router)` — registers /predict and /health

4. **Mount static files** (for single-Dockerfile architecture):
   - `app.mount("/", StaticFiles(directory="frontend", html=True), name="frontend")`
   - Mount AFTER `include_router` so API routes take precedence over static file serving
   - This serves `frontend/index.html` at the root path

5. **App metadata**: `FastAPI(title="Financial Risk API", lifespan=lifespan)`

**requirements.txt** — Add FastAPI and uvicorn:
