Create `backend/data/feature_engineering.py` with the following functions. This file already exists as an empty `__init__`-adjacent module, so create it fresh.

**Imports:** `numpy`, `pandas`, `typing` (Dict, List)

**Note:** The 9-feature set is: avg_income, avg_expenses, final_savings, debt_payment, credit_score, debt_ratio, liquidity_ratio, net_cash_flow, consec_negative_months. This replaces cf_volatility with net_cash_flow per locked decision to include all three derived features (debt ratio, liquidity ratio, net cash flow).

**Function 1: `compute_debt_ratio(avg_monthly_income: float, avg_monthly_expenses: float, monthly_debt_payment: float) -> float`**
- Returns `monthly_debt_payment / (avg_monthly_income - avg_monthly_expenses)` (debt relative to free cash flow, per locked decision)
- Returns `0.0` if `(avg_monthly_income - avg_monthly_expenses) <= 0` (avoids inf/NaN which breaks gradients)

**Function 2: `compute_liquidity_ratio(final_savings: float, avg_monthly_expenses: float) -> float`**
- Returns `final_savings / avg_monthly_expenses`
- Returns `0.0` if `avg_monthly_expenses <= 0`

**Function 3: `compute_net_cash_flow(avg_monthly_income: float, avg_monthly_expenses: float, monthly_debt_payment: float) -> float`**
- Returns `avg_monthly_income - avg_monthly_expenses - monthly_debt_payment` (per locked decision)
- No guard needed — negative values are valid and informative

**Function 4: `compute_cash_flow_volatility(monthly_cash_flows: List[float]) -> float`**
- Returns `float(np.std(monthly_cash_flows))` (standard deviation of monthly cash flows)
- Returns `0.0` if fewer than 2 data points
- NOTE: This function is kept in the file for completeness but is NOT used in the final 9-feature set

**Function 5: `compute_consecutive_negative_months(monthly_cash_flows: List[float]) -> int`**
- Iterates through cash flows, tracks current and max streak of negative values (cash_flow < 0)
- Returns max streak length as int

**Function 6: `engineer_features(profile_df: pd.DataFrame) -> Dict[str, float]`**
- Takes one profile's 12 rows (a DataFrame filtered to one profile_id)
- Sorts by `month` column to ensure correct ordering
- Computes `cash_flows = (income - total_expenses).tolist()` per row
- Extracts raw features:
  - `avg_income = float(group['income'].mean())`
  - `avg_expenses = float(group['total_expenses'].mean())`
  - `final_savings = float(group.iloc[-1]['savings'])` (month 12 row, after sorting)
  - `debt_payment = float(group.iloc[0]['debt_payment'])` (fixed per profile)
  - `credit_score = float(group.iloc[0]['credit_score'])` (fixed per profile)
- Calls `compute_debt_ratio(avg_income, avg_expenses, debt_payment)` (3-arg signature per locked decision)
- Calls `compute_liquidity_ratio(final_savings, avg_expenses)`
- Calls `compute_net_cash_flow(avg_income, avg_expenses, debt_payment)`
- Calls `compute_consecutive_negative_months(cash_flows)`
- Returns dict with all 9 keys in this exact order: `avg_income`, `avg_expenses`, `final_savings`, `debt_payment`, `credit_score`, `debt_ratio`, `liquidity_ratio`, `net_cash_flow`, `consec_negative_months`

**Function 7: `build_feature_matrix(df: pd.DataFrame) -> tuple[np.ndarray, np.ndarray, list[int]]`**
- Groups dataframe by `profile_id`
- For each group: calls `engineer_features()`, collects feature values as a list
- Also collects `is_stressed` label (from first row of each group) and profile_id
- Returns `(X, y, profile_ids)` where:
  - X: np.ndarray shape (n_profiles, 9) dtype float32
  - y: np.ndarray shape (n_profiles,) dtype float32
  - profile_ids: list of ints

**FEATURE_NAMES constant:** Define at module level: `FEATURE_NAMES = ['avg_income', 'avg_expenses', 'final_savings', 'debt_payment', 'credit_score', 'debt_ratio', 'liquidity_ratio', 'net_cash_flow', 'consec_negative_months']`

Add clear docstrings to each function explaining what it computes and why.
