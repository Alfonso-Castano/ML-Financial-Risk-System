  <name>Task 1: Implement evaluate.py with metrics and plotting functions</name>
  <files>backend/ml/evaluate.py</files>
  <action>
Create `backend/ml/evaluate.py` with the following functions:

**Imports:**
- `json`, `pathlib.Path`
- `numpy as np`, `torch`
- `matplotlib.pyplot as plt`
- `sklearn.metrics`: `recall_score`, `precision_score`, `f1_score`, `accuracy_score`, `roc_auc_score`, `confusion_matrix`, `roc_curve`

**Function: `compute_metrics(model, X_test: np.ndarray, y_test: np.ndarray, threshold: float = 0.5) -> dict`**
- Sets `model.eval()`
- Converts X_test to float32 tensor
- Uses `torch.no_grad()` for inference
- Gets probabilities: `model(tensor).squeeze().numpy()`
- Binary predictions: `(probs >= threshold).astype(int)`
- Computes and returns dict with:
  - `"recall": float(recall_score(y_test, preds))`
  - `"precision": float(precision_score(y_test, preds))`
  - `"f1": float(f1_score(y_test, preds))`
  - `"accuracy": float(accuracy_score(y_test, preds))`
  - `"roc_auc": float(roc_auc_score(y_test, probs))`
  - `"threshold": threshold`
- **Critical:** Wrap every sklearn metric with `float()` to avoid JSON serialization errors from numpy types

**Function: `save_metrics(metrics: dict, path: str) -> None`**
- Creates parent directory if needed
- Writes JSON with `indent=2`
- Prints metrics summary to console

**Function: `plot_loss_curves(train_losses: list, val_losses: list, path: str) -> None`**
- `plt.figure(figsize=(10, 6))`
- Plot train_losses and val_losses with labels
- xlabel: 'Epoch', ylabel: 'Loss', title: 'Training and Validation Loss'
- Legend, tight_layout, savefig with dpi=100
- **Always call `plt.close()` after saving** to release memory

**Function: `plot_confusion_matrix(y_true: np.ndarray, y_pred: np.ndarray, path: str) -> None`**
- Compute `cm = confusion_matrix(y_true, y_pred)`
- `fig, ax = plt.subplots(figsize=(6, 6))`
- `ax.imshow(cm, cmap='Blues')`
- Set tick labels: ['Healthy', 'Stressed'] on both axes
- xlabel: 'Predicted', ylabel: 'Actual', title: 'Confusion Matrix'
- Add text annotations in each cell: `ax.text(j, i, str(cm[i, j]), ha='center', va='center', fontsize=14)`
- Colorbar, tight_layout, savefig, `plt.close()`

**Function: `plot_roc_curve(y_true: np.ndarray, y_probs: np.ndarray, path: str) -> None`**
- Compute `fpr, tpr, _ = roc_curve(y_true, y_probs)`
- Compute `auc = roc_auc_score(y_true, y_probs)`
- `plt.figure(figsize=(8, 6))`
- Plot ROC curve with label `f'ROC Curve (AUC = {auc:.3f})'`
- Plot diagonal dashed line for random classifier
- xlabel: 'False Positive Rate', ylabel: 'True Positive Rate', title: 'ROC Curve'
- Legend, tight_layout, savefig, `plt.close()`

**Function: `run_evaluation(model, X_test, y_test, train_losses, val_losses, num_epochs_trained, split_info: dict) -> dict`**
- Convenience function that calls all the above functions in sequence
- Computes metrics
- Adds `split_info` entries to metrics dict (train_profiles, val_profiles, test_profiles, num_epochs_trained, input_size)
- Saves metrics to `settings.METRICS_PATH`
- Generates all 3 plots to `models/` directory
- Returns the full metrics dict
- Prints recall prominently: `"Test Recall: {recall:.4f} (target > 0.7)"`

**Use `matplotlib.use('Agg')` at the top of the file** (before `import matplotlib.pyplot`) to ensure non-interactive backend works on all systems (no display needed).
